# Changelog

Este projeto segue [Versionamento Semântico](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2026-04-08

### Adicionado
- Interface web com slider de cards para visualização de filmes
- Cadastro de filmes com capa, título, gênero, plataforma, ano, avaliação e review
- Avaliação com sistema de 10 estrelas
- Remoção individual de filmes
- Exportação da lista em formato `.json`
- Importação de lista previamente exportada
- Testes automatizados com pytest
- Linting com ruff
- Pipeline de CI com GitHub Actions
